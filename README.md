Frontend: Html, Css, Javascript, React.js, Tailwind, DaisyUI, Rechart, React-Toastify,React-Icon


Deployment: Vercel





### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohaiminul375/hero-io-0326
   cd hero-io-0326
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to start using the app.
